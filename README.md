# 三体 · 红岸 —— NDS 风格实时小剧场

改编自刘慈欣《三体》第一部《地球往事》的浏览器小剧场。纯前端、零依赖，用 Canvas 以
320×180 的像素分辨率实时演出一部三幕短剧：从叶文洁在红岸基地向太阳发出人类的第一声
呼唤，到三体世界监听员的抉择，再到那个改变人类命运的"回答"。

打开 `index.html` 即可运行。建议全屏、佩戴耳机。

## 操作

| 按键 | 作用 |
|------|------|
| `Z` / `空格` / `点击` | 推进对白 |
| `F` | 全屏 |
| `M` | 静音 |
| `R` | 重播 |

首屏需轻触/按键一次以解锁浏览器音频。

## 三幕流程

### 第一幕 · 红岸
大兴安岭雷达峰之巅，深夜飘雪。叶文洁走入巨型抛物面天线之下，发现太阳可作为天线的
放大器。她按下发射键，一束以恒星为放大器的电波射向太空。
- 场景：`redcoast`（雪夜天线）→ `control`（控制室）→ `sun`（太阳放大）
- 情绪：压抑、孤绝、越界前的犹疑
- 视觉点：极光、破晓渐变、发射时的馈源脉冲与电波环、太阳蓄能后放大射出

### 第二幕 · 三体世界
四光年外，被三颗太阳主宰的行星。文明已毁灭重生两百余次。第 1379 号监听员在乱纪元
（三日凌空）中目睹脱水与轮回，收到了来自地球的信号——一个稳定温暖的天堂。作为
和平主义者，他选择背叛自己的文明，向地球发出警告。
- 场景：`trisolaris`（三日凌空的荒漠 + 文明金字塔）
- 情绪：绝望的轮回、对安宁的渴望、悲悯的抉择
- 视觉点：三颗太阳各自游移，带灼热光晕与十字星芒；大地随乱纪元烤成炽红；
  监听员与卷起的脱水者

### 第三幕 · 回答
警告穿越四光年化作屏幕上的文字："不要回答！"。叶文洁面对这份来自宇宙的仁慈，却因
看透人类而做出相反的选择，按下按键，暴露了地球的坐标。
- 场景：`space`（深空星云）→ `control` → `space`（电波射向三体星）
- 情绪：宿命、人性的幽暗、"人类的落日"
- 视觉点：红/绿双色的警告文字逐条浮现、心跳蓄势、电波环射向远处的三体星系
- 尾声：星海铺底、星尘飘落，片尾滚动职员表（引言 + credits + 致谢）

## 音乐

动机取自霍尔斯特《行星组曲 · 火星，战争使者》（公有领域），标志性的 5/4 拍不祥固定
音型贯穿三幕做变奏。全部由 Web Audio 实时合成，无音频文件。

| 曲目 | BPM | 用于 | 基调 |
|------|-----|------|------|
| `act1` | 60 | 第一幕 | 压抑低回 |
| `act2` | 100 | 第二幕 | 全奏爆发（乱纪元） |
| `act3` | 66 | 第三幕 | 肃杀收束 |
| `title` | 60 | 片尾 | 星海挽歌 |

音效（实时合成）：脚步、打字提示音、心跳、电台/恒星电波、乱纪元警报、恒星轰鸣等。

## 技术结构

复用了一套与内容无关的通用引擎，内容层（场景/角色/配乐/剧本）为三体单独编写。

```
index.html            装配与脚本加载顺序
js/
  engine.js           数学/随机/颜色/像素绘制基元、内部分辨率常量
  display.js           双缓冲（像素层 + 高清 UI 叠层）、整数缩放、后处理
  audio.js            Web Audio 合成、音序器、音效（SFX）
  music.js            乐理工具（音名→频率）
  score.js / score2.js 四首乐曲的音轨数据
  scenes.js           五个场景的像素绘制（redcoast/control/sun/trisolaris/space）
  actors.js           角色像素画与分层明暗（叶文洁、监听员、脱水者、士兵等）
  ui.js               字幕、对话框、章节卡、标题卡、片尾滚动字幕、头像
  director.js         导演：镜头/灯光/调色/转场的状态与缓动
  play.js             第一幕剧本
  play2.js            第二、三幕剧本
  main.js             入口：状态机、主循环、输入、标题画面
```

剧本用 generator 协程编排：`wait`（等待秒数）、`box`（等待对白读完并推进）、
`tween`（缓动驱动镜头/灯光/参数）、`act`（即时动作）。导演对象（`S`）暴露
`setScene`、`cam`、`bright`、`vign`、`setGrade`、`bloom`、`music` 等，
剧本通过它们下达演出指令。

## 测试

`test/headless.js` 用 Node 的 `vm` 加载全部模块，桩掉 DOM/Canvas/RAF/定时器，
以 `skip` 模式加速跑完整部三幕，校验：

- 乐理频率（A4=440、音序正确）
- 四首乐曲音轨结构完整
- 五个场景全部渲染
- 剧情推进至 END（含片尾滚动字幕正常收尾）

运行：

```bash
node test/headless.js
```
